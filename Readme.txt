The files found in this github repository form the software program Interactive Dutch Weather Information System, referred to as IDWIS. The software provides yearly and monthly precipitation and temperature data of the Dutch weather stations. Different data can be obtained, visualised and compared with help of the graphs. Moreover, location information of the weather stations in the Netherlands can be obtained.
